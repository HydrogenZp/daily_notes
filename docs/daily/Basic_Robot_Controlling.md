## 闭环控制(Closed-loop control)

### Basic conception

闭环控制是一种需要被控对象反馈的控制方式，我们通过被控对象的反馈量可以计算出**误差**(``error``,简称 $\text{e}$)

在一些基础的控制算法中（如PID），得到误差后可以根据其计算出输出量
